01

はじめに

MUSCLE NAVIGATOR は、ジムに通わなくても自宅で本格的な筋力トレーニングとセルフケアができる Web アプリです。スマートフォン・タブレット・パソコンのブラウザで動作し、インストールは不要です。

このアプリでできること

  • 質問に答えるだけで自分の運動レベル（初級・中級・上級）を判定
  • 鍛えたい筋肉（全16部位）からトレーニングメニューを自動作成
  • 気になる症状（全15種）から改善・ケアのメニューを提案
  • 寝る前・朝・すきま時間など、目的別のストレッチ／セルフケア（全8ルーティン）
  • 好きな種目だけを集めて自分専用メニューを作成
  • メニューを選ぶ → タイマーで運動・休憩を自動管理 → 終わると日誌に自動記録
  • トレーニング履歴を記録し、トレ日数・総挙上量などを集計
ポイント：すべて自宅・器具なしでもOK

懸垂バーが必要な種目はありません。器具がない場合は「自重・器具なし」で絞り込めば、体ひとつでできる種目だけが表示されます。

動作環境と起動方法

  1. 配布された HTMLファイル（MUSCLE_NAVIGATOR_v5.html）をスマホやPCに保存します。
  2. ファイルをタップ／ダブルクリックすると、ブラウザでアプリが開きます。
  3. インターネット接続は不要です（オフラインで動作します）。
データの保存について

トレ日誌の記録は、お使いの端末のブラウザ内に保存されます。同じ端末・同じブラウザで開けば記録は残りますが、別の端末やブラウザの履歴削除を行うと記録は消えます。大切な記録は別途メモを残すことをおすすめします。

03

① 運動レベル判定

自分に合った難易度を知るための機能です。年齢や運動経験に加え、実際にできる回数を入力すると、初級・中級・上級を自動で判定します。

使い方

  1. 「①運動レベル判定」タブを開きます。
  2. 年齢を入力し、各質問で当てはまる選択肢をタップします（年齢以外はすべて回答が必要です）。
  3. 各設問のかっこ書きには正しいフォームのポイントが書かれています。その姿勢でできる回数を選んでください。
  4. 「レベルを判定する」を押すと、スコア（18点満点）と推奨難易度が表示されます。
  5. 「推奨レベルの種目を見る」を押すと、カスタマイズ画面に推奨難易度が引き継がれます。
懸垂が家でできない方へ

上級判定の指標は、自宅でもできる「パイクプッシュアップ（お尻を高く上げた逆Vの姿勢で頭を床に近づける腕立て）」を採用しています。懸垂バーは不要です。

判定の目安

初級：まず正しいフォームと習慣づくりの段階。週2〜3回の継続が目標。

中級：基礎ができている段階。中級種目を中心に、徐々に上級へ。

上級：高い基礎筋力あり。高難度の自重種目や高重量に挑戦できる。

※50歳以上の方は、安全のためスコアをやや控えめに補正します。

04

② 筋肉から探す

鍛えたい部位を選ぶと、その部位に効く種目を組み合わせたメニューを自動で作成します。複数の部位を同時に選ぶこともできます。

対応部位（全16部位）

胸背中肩力こぶ二の腕腹筋お尻太もも前太もも後ふくらはぎ僧帽筋体幹前腕手・指横腹首

使い方

  1. 鍛えたい部位をタップします（複数選択可。もう一度押すと解除）。
  2. 必要に応じて「絞り込み」で難易度と器具を指定します。
  3. 「メニューを表示する」を押すと、条件に合う種目が最大8件表示されます。
  4. 各カードをタップして、やり方・フォームのポイント・セット数・回数・休憩を確認します。

絞り込みの種類

項目選べる条件
難易度すべて / 初級 / 中級 / 上級
器具すべて / 自重・器具なし / 椅子・台 / ダンベル / チューブ
ヒント

絞り込みは結果表示後に変更すると、その場でメニューが更新されます。条件が厳しすぎて0件になった場合は、難易度や器具の指定をゆるめてください。

05

③ 症状から探す

肩こり・腰痛などの不調や、疲労感・冷えといった気になる状態から、改善やケアにつながる運動・ストレッチを提案します。

対応する症状（全15種）

肩こり腰痛膝の痛み猫背・姿勢不良足のむくみ股関節の硬さ緊張型頭痛手首・肘疲れ筋肉痛全身の疲労感手足の痺れふらつき眼精疲労冷え性寝つきが悪い

使い方

  1. 気になる症状をひとつ選びます。
  2. 「改善メニューを表示する」を押すと、原因の解説と一緒にケア種目が表示されます。
  3. 各カードで、やり方と注意点（⚠）を必ず確認してから行ってください。
安全に関する大切なお願い

手足の痺れ・ふらつきなどは、運動で和らぐこともありますが、症状が片側だけ・持続する・力が入らない・回転性のめまいや吐き気を伴う場合は、自己判断で運動を続けず、必ず医療機関を受診してください。

このアプリは医療行為の代わりではありません。痛みが強いとき、持病があるときは、医師に相談のうえご利用ください。

07

⑤ カスタマイズ

すべての種目の中から好きなものだけを集めて、自分専用のメニューを作る機能です。

使い方

  1. 検索ボックスに種目名や部位を入れる、または難易度のフィルタで絞り込みます。
  2. 追加したい種目の「＋」ボタンを押すと、画面上部の「MY MENU」に入ります。
  3. 「−」または「✕」で取り消せます。MY MENUには合計種目数と推定時間が表示されます。
  4. MY MENU の ▶ このメニューをタイマーで実行 を押すと、組んだメニューをそのままタイマーで開始できます。
レベル判定との連携

「①運動レベル判定」で推奨レベルを出してからこの画面を開くと、最初から推奨難易度で絞り込まれた状態になります。

08

ワークアウト & タイマー機能

選んだ種目を、タイマーで「運動 → 休憩 → 次のセット」と自動で進めてくれる機能です。運動が終わると自動で日誌に記録されます。

メニューの集め方（2通り）

  • その都度追加：どのタブでも、種目カードを開いて ＋ ワークアウトに追加 を押します。追加すると画面下にフローティングバーが現れ、追加した種目数が表示されます。スタート ▶ で確認画面へ。
  • カスタマイズから：「⑤カスタマイズ」で組んだ MY MENU の ▶ このメニューをタイマーで実行 を押すと、そのまま確認画面へ進みます。

確認画面でタイマー方式を選ぶ

開始前の確認画面で、種目の一覧を確認し、次の2つのタイマー方式から選べます。各方式の秒数はあらかじめ決まっています（変更不要）。いずれも3セットを自動で進めます。

方式1セットの流れセット間/種目間
① 通常式30秒運動 →（休憩）を3セット繰り返すセット間15秒 / 種目間60秒
② 3/7式3秒→4秒→5秒→6秒→7秒の保持（各保持の間に15秒インターバル）で1セットセット間15秒 / 種目間60秒
どちらを選べばいい？

① 通常式は、一定時間しっかり動く・キープするオーソドックスな方式です。腕立てやプランクなど多くの種目に向いています。

② 3/7式は、短い保持を少しずつ伸ばしながら繰り返す方式で、筋肉に効かせたいときや等尺性（キープ系）の種目に向いています。

流れの例

通常式：30秒運動 → 15秒休憩 → 30秒 → 15秒 → 30秒 →（種目間60秒）→ 次の種目へ。

3/7式：3秒保持 → 15秒 → 4秒 → 15秒 → 5秒 → 15秒 → 6秒 → 15秒 → 7秒 →（セット間15秒）→ 2セット目 → 3セット目 →（種目間60秒）→ 次の種目へ。

タイマー画面の操作

  • 中央の円が残り時間をカウントダウンします（運動中は黄色、保持＝3/7式のキープ中はオレンジ、休憩中は緑）。
  • 画面には「セット ◯/3」「◯本目」「◯秒キープ」など、今どこを行っているかが表示されます。
  • 運動・保持中は、その種目のフォームのポイントも表示されます。
  • フェーズの切り替わりは、音とバイブ（対応端末）でお知らせします。
  • 「⏸ 一時停止 / ▶ 再開」「⏭ スキップ」「■ 中止」で操作できます。
運動が終わると…

すべてのセットが終わると「WORKOUT COMPLETE」と表示され、実施した種目・セット数・選んだ方式が自動で「⑥筋トレ日誌」に記録されます（メモ欄に「タイマー（通常式／3/7式）で実施（自動記録）」と入ります）。「記録を見る」で日誌へ移動できます。

※「■ 中止」で途中終了した場合は記録されません。

09

⑥ 筋トレ日誌

トレーニングの記録をつけ、続けるほど積み上がる数字を確認できる機能です。タイマーからの自動記録に加え、手動でも入力できます。

手動で記録する

  1. 日付・種目名を入力します（種目名は候補から選べます）。
  2. 重量(kg)・回数・セット数・メモを必要に応じて入力します。
  3. 「記録する」を押すと履歴に追加されます。

集計される数字

指標内容
トレ日数記録のある日付の数
記録数登録された種目記録の合計件数
総挙上量(kg)重量 × 回数 × セット数 の合計
  • 各記録の「✕」で削除できます（削除すると元に戻せません）。
10

よくある質問・困ったとき

こんなとき対処
メニューに何も出ない筋肉タブで、絞り込みが厳しすぎると0件になります。難易度や器具を「すべて」に戻してください。
タイマーの音が鳴らない端末がマナーモードのことがあります。音量設定を確認してください。バイブは非対応の端末では動作しません。
記録が消えたブラウザの履歴・サイトデータを削除すると記録も消えます。別の端末・ブラウザでは共有されません。
下のバーが出ない種目を1つ以上「ワークアウトに追加」すると表示されます。「クリア」で空にできます。カスタマイズの MY MENU からは「▶ このメニューをタイマーで実行」で直接開始できます。
3/7式と通常式の違いは？通常式は30秒運動×3セット。3/7式は3→4→5→6→7秒の保持を15秒間隔で行うのを1セットとして3セット行います。開始前の確認画面で選べます。
休憩の秒数を変えたい各方式の秒数（運動・休憩・セット間・種目間）はあらかじめ決まっており、変更はできません。
器具がない器具の絞り込みで「自重・器具なし」を選べば、体ひとつでできる種目だけになります。
11

安全にお使いいただくために

  • 運動前は軽く体を動かして温め、痛みを感じたらすぐに中止してください。
  • 正しいフォームを最優先にし、無理な重量・回数・難易度は避けてください。
  • 持病がある方、妊娠中の方、痛みや不調が続く方は、医師に相談のうえご利用ください。
  • 手足の痺れ・強いめまい・力が入らないなどの症状があるときは、運動せず医療機関を受診してください。
  • 水分をこまめに補給し、体調がすぐれない日は休む勇気を持ちましょう。
